# Baby Ava's Arrival Prep Checklist

https://ee-edk.github.io/AvA-ToDO/

A clean, mobile-optimized checklist for tracking baby preparation tasks. Built with vanilla HTML/CSS/JS for maximum compatibility and performance.

## Features
- **Progress Tracking:** Interactive progress bar and task counter.
- **Visual Feedback:** Confetti celebrations for completed tasks and a final 100% burst.
- **Persistence:** Automatically saves your progress to `localStorage`.
- **Responsive Design:** Optimized for both desktop and mobile viewports.
- **Accessibility:** Built with ARIA roles and semantic HTML.

## Project Structure
- `src/`: Source code directory.
  - `index.html`: The main application file (v2.0).
- `docs/`: Deployment directory for GitHub Pages.
  - `index.html`: The production-ready file.
- `CLAUDE.md`: Development guidelines and project context.
- `GEMINI.md`: Project mandates for Gemini CLI.

## How to Use
1. Clone the repository.
2. Open `src/index.html` in your browser to start tracking tasks.
3. Your progress is automatically saved to your browser's local storage.

## Deployment
This project is hosted via **GitHub Pages** from the `docs/` directory.

To update the live site:
1. Make changes to `src/index.html`.
2. Sync the changes to `docs/index.html`.
3. Commit and push your changes to the `main` branch.
